Относительность
Движения

Всё в мире находится в движении, но как мы его измеряем? Погружение в фундаментальные принципы относительности Галилея.

Теоретическое обоснование

Система отсчета

Совокупность объектов, необходимых для описания движения тела в любой момент времени.

01. Тело

Относительно которого определяется положение.

02. Координаты

Обычно Декартова система (X, Y, Z).

03. Время

Прибор для измерения (Часы).

Принцип Галилея

Все механические явления протекают одинаково во всех инерциальных системах отсчета. Законы физики инвариантны.

"Никакими опытами внутри системы невозможно определить, покоится она или движется равномерно."

Математический аппарат

Преобразование координат

Связь радиус-векторов в неподвижной (S) и движущейся (S') системах:

\[\vec{r} = \vec{r'} + \vec{v}_{\text{отн}} \cdot t\]

Закон сложения скоростей

Главная формула для решения задач на относительность:

\[\vec{v} = \vec{v'} + \vec{v}_{\text{отн}}\]
  • \(\vec{v}\) — скорость тела относительно неподвижной системы
  • \(\vec{v'}\) — скорость тела относительно движущейся системы
  • \(\vec{v}_{\text{отн}}\) — скорость движущейся системы относительно неподвижной

Важное замечание

В классической механике Галилея время абсолютно (\(t' = t\)). Это означает, что интервалы времени и ускорение (\(\vec{a}\)) не меняются при переходе из одной инерциальной системы в другую.

Это приближение верно только для скоростей значительно меньше скорости света.

Практика: Пошаговые решения

Пример 1: Одномерное движение Автомобили

Автомобиль A (60 км/ч, восток) и B (40 км/ч, запад). Найти скорость B относительно A.

1
Выбор оси

Направим ось X на восток. Тогда \(v_A = 60\), \(v_B = -40\).

2
Формула

\(v_{B \text{ отн } A} = v_B - v_A\)

-100 км/ч

Отрицательный знак означает направление на запад

Пример 2: Двумерное движение Лодка на реке

"Лодка (4 м/с перпенд. берегу) на реке (3 м/с вдоль). Ширина 120 м."

Результирующая скорость
\(V = \sqrt{3^2 + 4^2} = 5 \, \text{м/с}\)
Время пересечения
\(t = 120 / 4 = 30 \, \text{с}\)
Смещение: \(X = V_{\text{течения}} \cdot t = 3 \cdot 30 = 90 \, \text{м}\).
Пример 3: Траектория в поезде Парабола vs Прямая

Мяч брошен вверх (\(10 \, \text{м/с}\)) в поезде (\(20 \, \text{м/с}\)). Описать движение относительно Земли.

\[x(t) = 20t\] \[y(t) = 1.5 + 10t - 4.9t^2\]
Уравнение траектории: \(y(x) = 1.5 + 0.5x - 0.01225x^2\)

Для пассажира мяч летит строго по вертикали. Для наблюдателя на перроне — по параболе. Это и есть относительность траектории.

Идеи для анимаций

Прозрачный вагон

Синхронное сравнение вида изнутри и снаружи поезда при броске мяча.

Векторный конструктор

Интерактивное сложение векторов лодки и течения с мгновенным пересчетом траектории.
