Первый закон термодинамики

Закон сохранения энергии гласит: энергия не берется из ниоткуда и не исчезает в никуда. В термодинамике этот великий принцип принимает свою самую знаменитую форму.

1. Формулировка Первого закона термодинамики

Мы уже знаем, что изменить внутреннюю энергию (ΔU) системы можно двумя способами: передав ей количество теплоты (Q) или совершив над ней механическую работу. Если мы подводим тепло к газу, куда оно расходуется?

Первый закон термодинамики: Количество теплоты (Q), переданное системе, идет на изменение ее внутренней энергии (ΔU) и на совершение системой работы (A') против внешних сил.

Q = ΔU + A'

Правило знаков (очень важно!):

  • Q > 0: система получает тепло; Q < 0: система отдает тепло.
  • ΔU > 0: газ нагревается; ΔU < 0: газ охлаждается.
  • A' > 0: газ расширяется (совершает работу); A' < 0: газ сжимают внешние силы.

Альтернативная запись: Если работу совершает не газ (A'), а внешние силы над газом (A), то A = −A'. Тогда первый закон запишется так: ΔU = Q + A.

2. Применение к изопроцессам

Если один из параметров (p, V, T) зафиксирован, первый закон заметно упрощается:

1. Изохорный процесс (V = const)

Так как объем не меняется (ΔV = 0), газ не может двигать поршень, следовательно A' = 0. Вся теплота идет только на нагрев:

Q = ΔU

2. Изотермический процесс (T = const)

Так как температура неизменна, внутренняя энергия тоже не меняется (ΔU = 0). Все переданное тепло целиком превращается в механическую работу (идеальная машина!):

Q = A'

3. Изобарный процесс (p = const)

Здесь меняются и объем, и температура. Подведенное тепло делится на две части: газ и нагревается, и расширяется.

Q = ΔU + A'

4. Адиабатный процесс (Q = 0)

Это процесс, протекающий в теплоизолированной системе (или очень быстро, так что газ не успевает обменяться теплом). Теплообмена нет, Q = 0. Следовательно, 0 = ΔU + A', или:

A' = −ΔU

Вывод: Если газ совершает работу адиабатно (расширяется), он делает это за счет своей внутренней энергии, поэтому газ охлаждается! (Пример: снежок из огнетушителя).

4. Пример решения задачи

Условие: Идеальному одноатомному газу передали количество теплоты Q = 500 Дж. При этом газ расширился при постоянном давлении (изобарно). Какую работу совершил газ? Как изменилась его внутренняя энергия?

Дано:
Q = 500 Дж
p = const
Найти: A' — ?, ΔU — ?

Решение:
Для изобарного процесса одноатомного газа есть жесткое соотношение. Вспомним, что:
ΔU = 3/2 νRΔT = 1.5 · pΔV.
А работа A' = pΔV.
Следовательно, ΔU = 1.5 · A'.

Запишем первый закон термодинамики:
Q = ΔU + A'
Q = 1.5 · A' + 1 · A' = 2.5 · A'.

Теперь легко найти работу:
A' = Q / 2.5 = 500 / 2.5 = 200 Дж.

А изменение внутренней энергии:
ΔU = Q - A' = 500 - 200 = 300 Дж.

Ответ: Газ совершил работу 200 Дж, а 300 Дж пошли на увеличение его внутренней энергии.
